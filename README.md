## What is iTravel?

iTravel is a travel companion app for Android. Browse **60 hand-picked destinations** across 30+ countries on an interactive map, check the weather, scroll through beautiful travel photos, and leave reviews when you actually visit a place â€” the app uses your phone's GPS to make sure you're really there.

Whether you're planning your next trip or reminiscing about your last one, iTravel puts the world at your fingertips.

> [!NOTE]
> **This is V2** â€” a complete MVVM rewrite with a new UI/UX, expanded content, and modern architecture. See [Project History](#project-history) for details.

## Getting Started

You'll need an **Android phone** running version 8.0 or newer (most phones from 2017+).

1. Download [Android Studio](https://developer.android.com/studio)
2. Clone the project:
   ```
   git clone --recurse-submodules https://github.com/HlexNC/iTravel.git
   ```
3. Open it in Android Studio and hit the green **Run** button
4. That's it â€” the app launches on your phone or emulator

> [!IMPORTANT]
> The map, travel photos, and weather features need free API keys to work. Without them the app still runs, but those sections will show placeholder content. See [docs/API_SETUP.md](docs/API_SETUP.md) for setup details.
